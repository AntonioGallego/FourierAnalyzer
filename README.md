# FourierAnalyzer
3D Fourier Signal Analyzer -- Interactive WebGL visualization tool that decomposes signals into their frequency components using Fourier series. Layers stack in 3D depth: individual sinusoidal harmonics receding into a rotatable cuboid. Square, sawtooth, triangle, pulse & custom waveforms. Adjustable harmonics, depth spacing & animation speed. Built with Three.js.
